#Oracle 11g安装

关于 ORACLE 下载

1、下载Oracle 11g R2 for Windows版本

2、下载完成后解压缩两个压缩包到同一个目录，即”database”

下方为ORACLE官网下载地址:

  • oracle 11g database1 win32
  • oracle 11g database2

ORACLE 安装

进入解压缩文件夹，找到”setup.exe”文件并双击

执行后会出现如下命令提示：

等待片刻后进入加载页面

出现安装界面，默认点击”下一步”

安装选项默认点击”下一步”

进入系统类选项，如果是个人笔记本使用选择“桌面类”，如果是安装在windows server上则选择”服务器类”

安装配置需要注意管理口令格式需要一个小写字母和一个数字，一个大写字母，否则会提示警告，路径默认，选择完成后点击”下一步”

先决条件确认，忽略全部，点击”下一步”

点击”完成”开始进行安装

开始安装数据库

数据库安装进度

安装完成后，在开始菜单中找到->Oracle - OraDb11g_home1->Database Control - orcl中打开访问网址

在连接身份里选择”SYSDBA”,再用户名处输入”sys”，密码为你最初设定的密码，点击”登陆”则可以访问你的数据库
